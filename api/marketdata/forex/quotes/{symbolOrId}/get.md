# Get Crypto Quote

Returns crypto quote data.

**URL** : `/marketdata/forex/quotes/{symbolOrId}`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Path constraints**

    - {symbolOrId} - UUID (for example, `cc0efa02-cfca-11e0-ae25-8cae28fccae2`) or symbol (for example, BTCUSD)

## Success Response

**Code** : `200 OK`

**Content examples**
