# Get Crypto Currency Pairs

Returns a paginated list of Crypto currency pairs.

**URL** : `/currency_pairs/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Query constraints**

    - query - optional search string
    - cursor - optional pagination cursor

## Success Response

**Code** : `200 OK`
