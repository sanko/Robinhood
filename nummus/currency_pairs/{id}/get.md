# Get Crypto Currency Pair by ID

Returns data on a single Crypto currency pair.

**URL** : `/currency_pairs/{id}/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Path constraints**

    - {id} - UUID (for example, `3d961844-d360-45fc-989b-f6fca761d511`)

## Success Response

**Code** : `200 OK`
